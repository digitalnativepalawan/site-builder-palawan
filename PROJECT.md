# Website Creator Webapp: Project Manifesto

## Purpose
A high-efficiency site builder enabling resort owners to transform raw data into a premium hospitality web presence.

## The User Journey
1. Identity Phase: Capture core resort metadata.
2. Media Phase: Curate high-impact visual assets.
3. Amenity Phase: Define features (e.g., Starlink, Solar, AC).
4. Generation: Automate the build.
5. Management: Preview and Dashboard control.

## Admin Capabilities
- Visuals: Hero sections, Video backgrounds, Image galleries.
- Structure: Global Header/Footer management.
- System: Technical settings and SEO.

## The Mission
To provide non-technical users with a "Vibe-First" creation tool that prioritizes clean aesthetics and speed over complex configuration.
